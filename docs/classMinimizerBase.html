inlineprotected

Perform a line search to find a suitable step length alpha.

This routine attempts to find a step length alpha along direction p starting from point x such that (approximate) Wolfe conditions are satisfied:

  • sufficient decrease condition (controlled by c1)
  • curvature condition (controlled by c2)
Parameters
xCurrent point.
pSearch direction.
fObjective function to minimize.
GradientFunction object returning the gradient of f.
Returns
Step length alpha found by the line search. If no suitable alpha is found within max_line_iters, the last tested alpha is returned as a fallback.

Solve the minimization problem given an initial guess.

This is the main entry point of any concrete minimization algorithm inheriting from this base class.

Parameters
xInitial guess for the minimizer; can be used as in/out.
fObjective function to minimize, mapping V -> double.
GradientFunction object returning the gradient of f, mapping V -> V.
Returns
Approximate minimizer of the function f.

Implemented in Newton< V, M >, LBFGS< V, M >, and BFGS< V, M, Solver >.
